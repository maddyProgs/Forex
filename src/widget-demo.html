Trading Widget Integration Demo

Basic Widget Implementation

This demonstrates how to embed the trading widget on any website:

<!-- Include the widget script -->
<script src="https://your-domain.com/widget/trading-widget.js"></script>

<!-- Add the widget element -->
<trading-widget client-id="your-client-id"></trading-widget>

Customized Widget

Widget with custom dimensions and settings:

<trading-widget
  client-id="your-client-id"
  width="800px"
  height="400px"
  title="Custom Trading Dashboard"
  show-header="true"
  show-footer="false"
  default-view="grid">
</trading-widget>

Minimal Widget

Compact widget without header and footer:

<trading-widget
  client-id="your-client-id"
  width="600px"
  height="300px"
  show-header="false"
  show-footer="false">
</trading-widget>

Integration Instructions

For Client Websites:

  1. Include the widget script in your HTML head or before closing body tag
  2. Add the trading-widget element where you want the widget to appear
  3. Configure the client-id and other attributes as needed
  4. The widget will automatically load and display trading signals

Available Attributes:

  • client-id: Your unique client identifier (required)
  • width: Widget width (default: "100%")
  • height: Widget height (default: "600px")
  • title: Widget title (default: "Trading Signals")
  • show-header: Show/hide header (default: true)
  • show-footer: Show/hide footer (default: true)
  • default-view: Initial view "grid" or "detail" (default: "grid")
